Detailed Description

This class provides methods and attributes to handle IENA Ethernetpackets.

This class provides IENA objects conform to IENA standard rev 6.0 It provides methods to increment the sequence number as well as methods to compute the header time and to set up the IENA payload

Note
KEEP IN MIND: W5500 uses Big Endian, where as Atmega32u4 uses Little Endian for word operations. For 16bit values, either define it already in Big Endian Order or doing a Byte Swapping
