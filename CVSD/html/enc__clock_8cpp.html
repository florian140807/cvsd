ISR ( TIMER1_COMPA_vect  )

Interrupt Service Routine (ISR) TIMER1 COMPA.

This ISR is going to be executed on TIMER1 compare match. The rate of the ISR is being called is set up by the enc_clock() constructor and can be either 16ksps, 32ksps or 64ksps.

At the selected speed the CVSD encoder data clock pin is triggered.

The ISR incorporates are ring buffer to share the encoder byte data with the main() loop

set encoder data clock pin

wait at least 1us before checking encoder output state (750ns min)

read encoder output state pin

shift enc_byte 1 to the left and write Encoder output into LSB position of enc_byte

decrement shift counter

reset encoder data clock pin; (Clock 1 Pulse Width minimum: 1us)

reset ready bit indicates encoder byte is not ready to be written in the buffer

check shift counter Value

all 8 bits placed in the encoder byte

write encoder byte into buffer, increment write counter

mask write counter value to avoid buffer running out of range

reset shift counter

tell main() that encoder byte is ready to be processed
